This dataset is based on MovieLens datasets (https://grouplens.org/datasets/movielens/) and contains user ratings assigned to movies, along with additional movie-related information.

The dataset consists of three files:
ratings.csv – user ratings
metadata.csv – information about movies, such as titles, directors and cast
tags.csv – number of times a certain tag has been attached to a movie

This dataset has been constructed for the following course: “Seminar: Topics in Recommender Systems, Seminar”. It is intended solely for educational purposes within the confines of this course. Please refrain from using this dataset outside the designated course or sharing it with others.
